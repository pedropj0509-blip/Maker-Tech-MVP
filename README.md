# Maker Tech - MVP Mobile App

> **Status:** Em Desenvolvimento  

> **Framework:** Flutter 3.41.9

> **Objetivo:** Criar um MVP (Produto Mínimo Viável) do aplicativo 
 Através da união entre *microlearning* e gamificação, o app reduz as barreiras iniciais de aprendizado, transformando conceitos complexos em desafios interativos.
---

## Tecnologias

### Core

- **[Flutter 3.41.9](https://flutter.dev/):** (Versão 3.41.9) Framework cross-platform para desenvolvimento mobile.
- **Dart:** Linguagem de programação orientada a objetos.
- **Backend as a Service (BaaS):** [Supabase](https://supabase.com/) (Autenticação, Banco de Dados Relacional e Storage) **(AINDA NÃO DISPONÍVEL)**
- **Gerenciamento de Estado:** [Riverpod](https://riverpod.dev/) **(AINDA NÃO DISPONÍVEL)**
- **Arquitetura:** MVVM (Model-View-ViewModel)
